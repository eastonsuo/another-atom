## 配额口径

计量单位为实际 Provider 请求次数，token 作为审计字段记录。结构化修复若触发第二次 Provider 请求会结算两单位；未发生的预占会释放。失败 Run 不再把全部剩余预占计为已用，但真实发生的失败/重试请求仍计量。

## 当前保留边界

- Blueprint 生成仍同步阻塞 `POST /runs`，尚未改为 202 + SSE。
- Build Worker 当前持久化执行 AppSpec 校验和版本生成；每项目独立源码物化与 `npm run build` 尚未实现，README 不将其标记为已完成。
- Deployment 当前是一项目一条活动记录；事件可以审计动作，但不是完整发布历史模型。
- V1 单实例 Worker lease 是当前部署基线；水平扩展前仍需拆分 Worker 并迁移共享产物存储。
